# 机器学习理论面试精要

## 核心概念

### 1. 监督学习与无监督学习
- 监督学习：利用带标签的数据训练模型，预测输出（如分类、回归）。
- 无监督学习：处理无标签数据，寻找数据内部结构（如聚类、降维）。

### 2. 过拟合与欠拟合
- 过拟合 (Overfitting)：模型在训练集表现极好，但在测试集表现差。解决方法：正则化、Dropout、增加数据、早停。
- 欠拟合 (Underfitting)：模型无法捕捉数据规律。解决方法：增加模型复杂度、特征工程。

### 3. 偏差-方差权衡 (Bias-Variance Tradeoff)
- 偏差 (Bias)：模型预测值与真实值之间的误差，反映模型拟合能力。
- 方差 (Variance)：模型在不同训练集上表现的波动，反映模型稳定性。
理想模型是低偏差、低方差。

## 模型评估

### 4. 分类指标
- 准确率 (Accuracy)：容易受类别不平衡影响。
- 精确率 (Precision) 与 召回率 (Recall)：精确率关注“找得准”，召回率关注“找得全”。
- F1-Score：精确率与召回率的调和平均。
- ROC-AUC：衡量模型排序能力，对类别不平衡不敏感。

### 5. 回归指标
- MSE (均方误差) / RMSE：对异常值敏感。
- MAE (平均绝对误差)：对异常值更具鲁棒性。
- R-squared：衡量模型解释数据方差的比例。

## 常用技术

### 6. 特征工程
数据和特征决定了机器学习的上限。包括：缺失值处理、特征缩放 (Normalization/Standardization)、独热编码 (One-hot)、特征选择。

### 7. 交叉验证 (Cross-Validation)
通过将数据划分为多个折，重复训练和验证，以获得模型性能的稳定估计，防止过拟合。

### 8. 正则化 (Regularization)
- L1 正则化 (Lasso)：产生稀疏特征，可用于特征选择。
- L2 正则化 (Ridge)：防止权重过大，增强模型泛化能力。
