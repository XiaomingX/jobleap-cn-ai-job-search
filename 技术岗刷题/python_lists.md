# Python 数据容器：列表、字典与字符串

## 列表 (Lists)
- 特点：可变序列，支持切片。
- 常用操作：`append()`, `extend()`, `pop()`, `sort()`。
- 切片操作：`list[start:stop:step]`。

## 字典 (Dictionaries)
- 特点：键值对映射，无序（3.7+ 保持插入顺序），查找效率高 (O(1))。
- 常用操作：`get()` (防报错), `keys()`, `values()`, `items()`。
- 字典推导式：`{k: v for k, v in ...}`。

## 字符串 (Strings)
- 特点：不可变序列。
- 格式化：推荐使用 f-strings (`f"{var}"`)。
- 常用操作：`strip()`, `split()`, `join()`, `replace()`。
