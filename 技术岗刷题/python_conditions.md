# Python 流程控制与循环实战

## 条件判断 (Conditions)
- `if-elif-else` 结构。
- 真值测试：空容器 ([], {}, "")、0、None 均为 False。
- 逻辑运算符：and, or, not (短路计算)。

## 循环控制 (Loops)
- `for` 循环：主要用于遍历容器或迭代器。
- `while` 循环：用于满足特定条件时的重复执行。
- `break` 与 `continue`：提前跳出循环或跳过当前迭代。
- `enumerate()`：同时获取索引和值。
- `zip()`：并行遍历多个容器。
