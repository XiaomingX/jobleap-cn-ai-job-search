# Excel 数据分析精要

## 核心函数
- VLOOKUP / XLOOKUP: 数据查找与关联。
- INDEX + MATCH: 灵活的查找组合。
- IF / IFS: 逻辑判断。
- SUMIFS / COUNTIFS: 条件求和与计数。

## 核心功能
- 数据透视表 (Pivot Tables): 快速汇总与交叉分析。
- 动态数组 (Dynamic Arrays): 如 UNIQUE, SORT, FILTER 函数。
- Power Query: 数据清洗与转换。
