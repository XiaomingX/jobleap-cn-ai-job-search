# Python 数据分析全流程

## 1. 数据获取
- 读取 CSV/Excel：`pd.read_csv()`, `pd.read_excel()`。
- 数据库连接：`sqlalchemy`, `pandas.read_sql()`。

## 2. 探索性数据分析 (EDA)
- 快速概览：`df.info()`, `df.describe()`, `df.head()`。
- 缺失值查看：`df.isnull().sum()`。

## 3. 数据清洗与预处理
- 处理缺失：插值、删除。
- 异常值检测：3 Sigma、IQR 准则。
- 特征转换：标准化、归一化、分类变量编码。

## 4. 可视化分析
- 趋势分析：折线图。
- 关联分析：散点图、热力图 (Heatmap)。
