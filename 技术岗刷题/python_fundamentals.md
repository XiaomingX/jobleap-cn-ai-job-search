# Python 核心基础实战指南

## 核心语言特性

### 1. 动态类型与强类型
Python 是动态类型语言（运行时确定类型），同时也是强类型语言（不会发生隐式的类型转换，如 '1' + 1 会报错）。

### 2. 可变对象与不可变对象
- 不可变对象：int, float, bool, str, tuple。修改时会创建新对象。
- 可变对象：list, dict, set。修改时地址不变。

### 3. 深拷贝与浅拷贝 (copy vs deepcopy)
- 浅拷贝：只拷贝父对象，不拷贝内部子对象（子对象共享引用）。
- 深拷贝：完全拷贝父对象及其所有子对象，两者互不影响。

---

## 进阶语法

### 4. 装饰器 (Decorators)
本质是一个闭包，用于在不修改原函数代码的前提下增加新功能（如日志记录、性能测试）。

### 5. 生成器 (Generators)
通过 `yield` 关键字创建，每次返回一个值并保留当前状态。节省内存，适合处理大数据流。

### 6. 上下文管理器 (Context Managers)
使用 `with` 语句，自动管理资源（如文件关闭、数据库连接释放），确保代码安全性。

---

## 性能与最佳实践

### 7. 列表推导式 (List Comprehensions)
更简洁且通常比传统 for 循环执行效率更高。

### 8. 全局解释器锁 (GIL)
由于 GIL 的存在，Python 多线程无法利用多核 CPU。对于 CPU 密集型任务，应使用 `multiprocessing`。
