# NumPy 科学计算精要

## 核心概念
- ndarray：高性能多维数组对象。
- 向量化 (Vectorization)：消除显式 for 循环，利用底层 C 优化。
- 广播 (Broadcasting)：不同形状数组间的算术运算规则。

## 常用操作
- 创建：`np.array()`, `np.zeros()`, `np.linspace()`。
- 统计：`np.mean()`, `np.std()`, `np.sum(axis=0)`。
- 形状变换：`reshape()`, `flatten()`, `transpose()`。
