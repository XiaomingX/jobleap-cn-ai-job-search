# Python 字符串处理精要

## 文本处理
- `s.strip()`: 去除首尾空格。
- `s.lower() / s.upper()`: 大小写转换。
- `s.split(sep)`: 字符串切分。
- `", ".join(list)`: 高效连接字符串。

## 格式化
- 推荐：`f"value: {var}"` (f-string)
- 备选：`"value: {}".format(var)`

## 正则表达式
- 使用 `import re`。
- `re.search()`, `re.findall()`, `re.sub()`。
