### 一、 常见的大模型预训练与微调数据集

#### 1. 通用预训练文本语料（英文/多语言）
* **FineWeb / FineWeb-Edu** (Hugging Face 开源)
  * **特点**：目前开源质量最高的英文及教育类网页数据集之一，包含高达 15T Tokens 的清洗网页数据与高质量教育数据。
* **RedPajama / RedPajama-V2** (Together AI 开源)
  * **特点**：旨在复刻 LLaMA 训练集，包含网页（Common Crawl）、维基百科、图书、学术论文（ArXiv）、StackExchange 等，规模超过 30T Tokens。
* **SlimPajama** (Cerebras 开源)
  * **特点**：对 RedPajama 进行极致去重和清洗后的 627B Tokens 优质子集。
* **The Pile** (EleutherAI 开源)
  * **特点**：经典的多样化开源语料集（约 800GB），包含 22 个子数据集（涵盖学术、代码、图书、对话等）。
* **Dolma** (AI2 / 艾伦人工智能研究所开源)
  * **特点**：用于 OLMo 模型的 3T Tokens 开放透明预训练语料库。

#### 2. 中文预训练语料
* **书生·万卷（Wanjuan-CC / Wanjuan 1.0）** (上海人工智能实验室)
  * **特点**：包含数 TB 级别的高质量中文文本、图文交织及代码数据。
* **SkyPile-150B** (昆仑万维 / 天工)
  * **特点**：包含 1500 亿 Tokens 的纯净中文网页预训练数据集。
* **MNBVC (超大规模中文文本语料库)** (开源社区项目)
  * **特点**：目标对标 Common Crawl 的超大规模中文开源语料，包含新闻、网文、问答、百科等。
* **WuDao 2.0 (悟道语料)** (智源研究院 BAAI)
  * **特点**：高质量大规模中文文本语料。

#### 3. 代码语料数据集
* **The Stack / The Stack v2** (BigCode / Hugging Face)
  * **特点**：目前最主流的开源代码数据集，涵盖 600+ 种编程语言，包含宽松开源协议（Permissive）过滤版本。

#### 4. 指令微调与对齐数据集（SFT / RLHF）
* **UltraFeedback / UltraInteract**：高质量偏好对齐数据集（用于 DPO/RLHF）。
* **Alpaca / ShareGPT**：经典单轮及多轮对话指令数据集。
* **COIG (Chinese Open Instruction Generalist)**：智源等发起的中文指令微调数据集。

### 二、 在哪里可以下载到？

1. **Hugging Face Datasets**（全球最主流）
   * **网址**：`https://huggingface.co/datasets`
   * **使用方法**：直接在搜索框搜索数据集名称（如 `HuggingFaceFW/fineweb`、`bigcode/the-stack-v2`），支持使用 Python 库 `datasets` 一键加载/下载。
2. **OpenDataLab（浦源数据开放平台）**（国内首选，下载快）
   * **网址**：`https://opendatalab.org.cn`
   * **特点**：由上海人工智能实验室维护，托管了“万卷”等大量中文及多模态开源大模型数据集，提供国内极速下载。
3. **ModelScope（魔搭社区）**
   * **网址**：`https://modelscope.cn/datasets`
   * **特点**：阿里发起的开源社区，提供丰富的中文及多语言预训练/指令数据集，国内访问速度极快。
4. **Common Crawl 官网**（原始网页根源）
   * **网址**：`https://commoncrawl.org`
   * **特点**：全球最大的开放网页抓取数据库，提供原始的 WARC/WAT/WET 格式文件（需自行清洗与去重）。
