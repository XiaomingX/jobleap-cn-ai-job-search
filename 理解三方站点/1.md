简单来说，**Alienvault.com**（现已被 AT&T Cybersecurity 收购，品牌名为 LevelBlue）是一个网络安全平台。它既提供**完全免费**的社区工具，也提供**价格昂贵**的企业级商业方案。

根据你的需求，这里是其核心产品的使用方式和收费情况：

### 1. 它是免费的吗？

**结论：既有免费版，也有昂贵版。**

* **免费产品：**
* **OTX (Open Threat Exchange)**：完全免费。这是全球最大的威胁情报共享社区，任何人都可以注册并查询恶意的 IP、域名或文件哈希值。
* **OSSIM**：完全免费且开源。它是企业级 SIEM（安全信息和事件管理）系统的开源版本，适合预算有限的安全实验室或小型内网环境。


* **收费产品：**
* **USM Anywhere / USM Appliance**：非常昂贵。这是针对企业设计的托管式安全平台，起步价通常在 **$1,075/月**（约人民币 7,800 元/月）左右，根据数据量和资产数量价格会进一步飙升。

### 2. 主要功能及如何使用？

#### **如果你是个人安全爱好者（使用 OTX）**

* **用途**：查询某个 IP 是不是黑客地址，或者某个文件是不是病毒。
* **如何使用**：
1. 访问 [otx.alienvault.com](https://otx.alienvault.com/)。
2. 在搜索框输入你想检测的 IP 地址、域名或文件哈希（SHA256）。
3. 查看 **"Pulses"**（脉冲报告），它会告诉你这个威胁是谁发现的、属于什么攻击类型。



#### **如果你是企业管理员/运维（使用 OSSIM）**

* **用途**：监控公司网络里的入侵行为、漏洞扫描和日志管理。
* **如何使用**：
1. 下载 OSSIM 的 ISO 镜像并安装在虚拟机（如 VMware）上。
2. 配置网络流量镜像（Mirror Port），让 OSSIM 能够“看到”网络里的流量。
3. 通过 Web 界面查看安全警报。



#### **如果你是开发者（使用 API）**

* **用途**：把威胁情报集成到你自己的程序或防火墙中。
* **如何使用**：
1. 在 OTX 个人设置里获取 **API Key**。
2. 利用 Python 或提供的 SDK 调用接口，实现自动屏蔽恶意 IP。
