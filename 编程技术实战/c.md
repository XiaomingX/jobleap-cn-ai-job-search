# C/C++ 开发实战指南

## 核心能力与面试重点
C/C++ 作为底层技术的基石，其面试更看重候选人的“内功”。对内存管理的理解、系统底层知识以及对现代 C++ 特性的熟练运用是核心竞争力。

### 1. 核心复习要点
- 语言基础：C++17/20 新特性，STL 源码实现原理。
- 内存管理：RAII 机制，智能指针原理与避坑，内存泄漏排查。
- 并发编程：多线程同步原语，无锁队列，Atomic 操作。
- Linux 系统：IO 多路复用 (epoll), 信号处理, 进程间通信。

### 2. 简历提升建议
- 避免陈述：精通 C++，熟练掌握语法。
- 建议陈述：熟悉 C++11/17 特性，习惯应用 RAII 与智能指针管理资源，对 epoll 的 ET 模式有深度实践。
- 核心词汇：Reactor 模型, 零拷贝技术, 性能 Profiling, 模板元编程。

### 项目经历

#### 高性能异步网络服务器
- 情况：传统同步服务器在高并发下性能受限，资源利用率低。
- 任务：开发一套支持万级并发的轻量级 Web 服务器。
- 行动：
    - 采用 Reactor 模式，基于 Epoll 的边缘触发 (ET) 模式实现高效事件处理。
    - 实现基于任务队列的半同步/半异步线程池，优化并发处理能力。
    - 利用有限状态机高效解析 HTTP 协议，引入智能指针确保内存安全。
- 结果：单机并发处理能力达到万级，QPS 表现优异。
