# 软件工程师通用能力实战指南

## 核心底层与通用素质
技术栈可能会变，但底层逻辑是通用的。大厂面试中，基础能力的扎实程度往往决定了候选人的上限。

### 1. 通用核心能力
- 计算机基础：算法与数据结构 (LeetCode 300+), 操作系统, 计算机网络。
- 软件工程：设计模式, 代码质量, 单元测试, 持续集成。
- 解决问题能力：分析需求, 定位 Bug, 系统架构思维。
- 学习与协作：快速上手新技术, 团队沟通, 文档撰写。

### 2. 简历提升建议
- 避免陈述：熟悉各种编程语言。
- 建议陈述：具备扎实的计算机底层功底，能够灵活运用各种设计模式优化代码结构，有活跃的开源社区参与经验。
- 核心词汇：全栈思维, 性能调优, 自动化测试, 技术博客。

    - 使用 Python 构建分布式爬虫，通过代理池突破反爬限制。
    - 利用 Pandas 处理海量原始数据，并引入简单 NLP 算法进行情感分析。
    - 搭建可视化看板，为学校提供决策参考。
- 结果：日均处理数据万级，系统运行稳定。
