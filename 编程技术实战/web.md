# Web 前端开发实战指南

## 核心趋势与能力要求
当前前端开发已从单纯的页面渲染转向复杂的工程化实践。面试官不再关注基础的标签使用，而是考察对框架原理、性能优化及现代工具链的理解。

### 1. 技术栈建议
- 核心框架：React (18+) / Vue 3。
- 构建工具：Vite / Turbo / Rspack。
- 开发语言：TypeScript (必须熟练掌握类型体操)。
- 样式方案：Tailwind CSS / CSS Modules。

### 2. 简历避坑与提升
- 避免陈述：熟练使用 Vue/React 进行开发。
- 建议陈述：基于 Vue 3 + Vite 重构老旧项目，设计封装组件库，利用 Web Vitals 监控并将 LCP 降低 40%。
- 核心词汇：RSC (React Server Components), SSR/SSG/ISR, 微前端 (Micro Frontends), 虚拟列表, 骨架屏。

## 简历模板（核心片段）

### 技能清单
- 核心基础：深入理解 JavaScript (ES6+), CSS3 (Flex/Grid), HTML5。
- 框架生态：精通 React / Vue 3 生态，掌握 Zustand/Pinia 状态管理。
- 工程化：熟悉 Webpack/Vite 深度配置，掌握 CI/CD 自动化构建流程。
- 全栈潜力：熟悉 Node.js (NestJS) 开发，能够编写高效的 BFF 层。

### 项目经历

#### 某大型低代码可视化平台
- 情况：传统后台开发效率低，UI 风格不统一。
- 任务：设计实现可拖拽的低代码编辑器。
- 行动：
    - 基于 JSON Schema 定义组件协议，实现组件动态渲染。
    - 实现基于快照机制的撤销/重做功能，优化内存占用。
    - 采用虚拟滚动技术处理超长组件列表，确保渲染性能。
- 结果：提升团队开发效率 50% 以上。
