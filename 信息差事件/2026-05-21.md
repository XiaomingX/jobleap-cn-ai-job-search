  推荐的 AI Agent SaaS 平台（有免费层，中国尚未广泛讨论）

  1. Gumloop — AI 工作流自动化

  - 免费层: 2,000 credits/月，无限节点，可接入任意 LLM
  - 亮点: 自然语言构建 Agent 工作流，支持 MCP 服务器集成，无需自带 API Key
  - 适合: 非技术人员快速搭建自动化流程
  - 🔗 gumloop.com

  2. Taskade Genesis — AI Agent + 应用构建器

  - 免费层: 3 个可部署应用、1 个 AI Agent、3 个自动化工作流、3,000 AI credits
  - 亮点: 免费层包含真正的 AI Agent（不是简单 chatbot），可分享 URL，无需信用卡
  - 适合: 个人创业者快速搭建 AI 驱动的小应用
  - 🔗 taskade.com

  3. Activepieces — 开源自动化 + AI Agent 平台

  - 免费层: 开源自部署免费，云版有免费层
  - 亮点: 连接 450+ 服务，可视化构建 Agent 工作流，支持营销/运营/销售场景
  - 适合: 需要深度集成现有业务系统的团队
  - 🔗 activepieces.com

  4. n8n — 开源工作流自动化（自部署免费）

  - 免费层: 自部署完全免费，云版 $20/月起（14天免费试用）
  - 亮点: 1300+ 集成，可视化编辑器，支持接入任意 LLM（OpenAI/Claude/Gemini/本地模型）
  - 适合: 技术团队构建复杂 Agent 工作流
  - 🔗 n8n.io

  5. CrewAI — 多 Agent 协作框架

  - 免费层: Basic 计划免费，50 次工作流执行/月，可视化编辑器
  - 亮点: 专为多 Agent 协作设计，角色化 Agent 编排，代码量极少
  - 适合: 需要多个 AI Agent 协同完成复杂任务的场景
  - 🔗 crewai.com

  6. Flowise — 可视化 LLM Agent 构建器

  - 免费层: 自部署免费，云版 2 个流程 + 100 次预测/月
  - 亮点: Apache 2.0 开源，拖拽式构建 RAG 和 Agent 管道，支持自部署
  - 适合: 想快速搭建 RAG 应用和 Agent 的开发者
  - 🔗 flowiseai.com

  7. Langflow — 可视化 + 代码混合的 Agent 构建器

  - 免费层: 云版有免费层，自部署完全免费（MIT 协议）
  - 亮点: 既支持可视化拖拽，也支持 Python 代码自定义，适合有编程能力的开发者
  - 适合: Python 开发者构建复杂的 Agent 逻辑
  - 🔗 langflow.org

  8. Botpress — 企业级聊天 Agent 平台

  - 免费层: 免费使用，Team 计划 $495/月
  - 亮点: LLM 无关，190+ 集成，可视化构建器，支持自部署
  - 适合: 构建客服/对话类 AI Agent
  - 🔗 botpress.com

  9. Voiceflow — 对话式 AI Agent 设计平台

  - 免费层: 免费使用，Pro $60/月/编辑者
  - 亮点: 可视化画布，协作式原型设计，支持通过 Twilio 接入语音
  - 适合: 设计团队构建对话式 Agent
  - 🔗 voiceflow.com

  10. Dapti — AI Agent 构建平台（2026 新兴）

  - 免费层: 有免费层
  - 亮点: 专注 SMB 市场，4000+ 应用集成，100+ 预制 Agent 模板
  - 适合: 小微企业快速部署业务 Agent
  - 🔗 dapta.ai
