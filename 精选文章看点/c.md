### 💻 项目经历

#### 🚀 **高性能 HTTP 服务器**
**项目描述：**
仿照 Nginx 架构设计与实现的轻量级 Web 服务器，支持静态资源解析和高并发连接。

**我的贡献（STAR）：**
*   **Situation：** 需要一个深入理解网络编程和高并发模型的练手项目。
*   **Task：** 从零实现一个高性能、支持并发的 HTTP Server。
*   **Action：**
    *   采用 **Reactor** 模式，基于 **Epoll** (ET模式) 实现 IO 多路复用。
    *   实现 **线程池 (Thread Pool)** 处理业务逻辑，避免线程频繁创建销毁的开销。
    *   利用 **有限状态机 (Finite State Machine)** 解析 HTTP 请求报文。
    *   基于 **RAII** 机制封装互斥锁和条件变量，确保线程安全。
*   **Result：** 在 webbench 压测下，单机支持 **1w+** 并发连接，QPS 达到 Nginx 的 80%。
