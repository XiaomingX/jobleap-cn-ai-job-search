# Web 前端简历避坑：别让“切图仔”思维毁了你

## 前端已死？别制造焦虑了
这两年“前端已死”的言论满天飞。
其实死的不是前端，而是只会写 `div` + `css` 的“切图仔”。
校招面试官想看的，是你对 **工程化**、**性能优化**、**框架原理** 的理解。

## 💡 学长/学姐的修改建议
### 1. 别把“熟练使用 Vue/React”当成亮点
现在谁不会 Vue/React？这叫标配，不叫亮点。
❌ **Before：**
“熟练使用 Vue2 全家桶进行项目开发。”

✅ **After：**
“基于 Vue3 + Vite 重构老旧项目，设计封装通用组件库，将首屏加载速度提升 40%。”（体现工程化能力）

### 2. 性能优化是必考题
简历里一定要有 performance 相关的关键词：骨架屏、虚拟列表、Lazy Load、Webpack/Vite 构建优化等。
