# 个人简历撰写全攻略：从结构到细节

一份优秀的简历是打开理想工作大门的关键，它不仅是个人经历的陈述，更是展示你能力、职业态度和潜力的敲门砖。

## 一、简历的核心原则

1. **针对性**：根据目标岗位定制内容，突出匹配度最高的经历。
2. **量化成果**：用具体数字支撑成就，如“提升销售额20%”。
3. **结构清晰**：合理分配篇幅，重点突出工作/项目经历。
4. **简洁专业**：控制在一页纸内，使用 PDF 格式，无错别字。

## 二、简历的标准结构

### 1. 基本信息
- 姓名、电话、邮箱、求职意向。
- 居住地（可选，建议写到区）、GitHub/博客链接（技术岗推荐）。

### 2. 个人摘要
用 3-5 句精炼的话总结你的核心竞争力和职业亮点。

### 3. 工作经历与项目经验
- 采用逆时间顺序排列。
- 描述公式：动词 + 任务内容 + 方法/工具 + 结果（数据）。
- 重点写“你做了什么”以及“产生的价值”。

### 4. 教育背景
- 最高学历、学校、专业、起止时间。
- 应届生可标注 GPA 或核心课程。

### 5. 技能与荣誉
- 技能按掌握程度排序：精通 > 熟悉 > 了解。
- 荣誉奖项应与工作性质相关。

## 三、HR 评估简历的逻辑

HR 通常在 10-30 秒内决定一份简历的去留。他们主要寻找：
- **匹配度**：你的技能是否能解决岗位当前的痛点。
- **稳定性**：过往跳槽频率是否合理。
- **专业度**：简历的排版、用词、逻辑是否体现了职业素养。
