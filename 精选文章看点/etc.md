# 通用程序员简历：万变不离其宗

## 不限语言？其实看的是“内功”
有很多岗位写着“语言不限”，或者你是写 Python/Go/Ruby/Rust 等相对小众语言的。
这时候，面试官看的就不是 API 熟练度了，而是你的 **计算机基础**、**工程素养** 和 **解决问题的能力**。
这其实也是所有大厂校招的底层逻辑。

## 💡 学长/学姐的修改建议
### 1. 计算机基础是硬通货
不管你用什么语言，**数据结构与算法**、**计算机网络 (TCP/IP)**、**操作系统 (Linux)**、**数据库原理** 这些东西是通用的。
简历里必须体现这些“内功”。

### 2. 软技能 (Soft Skills) 也很重要
沟通能力、学习能力、团队协作……这些不是空话。
通过你在校学生会、开源社区、黑客马拉松 (Hackathon) 的经历来证明。
