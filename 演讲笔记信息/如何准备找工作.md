# 26届和27届应届生如何找工作

> 当前就业形势严峻，但**信息差**和**执行力**仍然是决定结果最关键的两个变量。

---

## 一、认清现状，找准定位

**两个核心问题先想清楚：**

1. **我能做什么？** ——技术栈、项目经验、实习经历、竞赛奖项
2. **市场要什么？** ——用招聘平台看真实 JD，而不是凭感觉猜

当下主流去向分三类：

| 类型 | 代表方向 | 竞争烈度 |
|------|----------|----------|
| 大厂互联网 | 算法、后端、数据、产品 | 极高 |
| 垂直行业 | 医疗AI、金融科技、工业软件 | 中等，专业匹配者优势大 |
| 央国企/事业单位 | 数字化岗位、IT岗 | 稳定，时间窗口短 |

> 💡 找准1个主赛道 + 1个保底赛道，不要海投散弹枪。

---

## 二、求职节奏（时间线非常关键）

许多同学错失机会，不是因为实力不够，而是**时间节点踩晚了**。

```
校招核心节奏（以9月入学的应届生为例）：

3月–5月   → 暑期实习投递高峰（优先争取，可能直接留用）
6月–8月   → 完成实习 + 刷题备战
7月–8月   → 提前批开放（很多公司早于官方批次）
9月–10月  → 秋招正式批（最密集，决战阶段）
11月–12月 → 补录 + 春招准备
```

**实习的重要性被严重低估：** 头部公司的实习转正录取率通常高于社招，且大量 offer 在正式秋招前已锁定。

## 三、简历：5分钟内抓住HR

**格式原则：**
- 一页纸，PDF格式，中英文按岗位选择
- 不用图片、不用颜色花哨的模板

**内容优先级（从上到下）：**

1. **项目经验** ——用数字量化（"准确率提升12%"比"效果好"有力十倍）
2. **实习经历** ——无实习则放竞赛/开源贡献
3. **技能栏** ——与JD关键词对齐，减少废话
4. **教育背景** ——学校/专业/GPA（后者仅限优秀时写）

> 一个常见误区：把简历写成"经历流水账"，而非"能力证明"。每条描述用 **做了什么 → 用什么方法 → 产生了什么结果** 的结构来写。

## 四、投递策略：精准 > 数量

### 信息获取是第一步

很多同学反馈：**不知道哪些公司在招、岗位要求是什么、薪资水位在哪**。

这里推荐一个专门聚合AI/科技方向校招岗位的平台 —— **[jobleap.cn](https://jobleap.cn)**，它会持续收录应届生友好的职位信息，尤其适合技术背景的同学做岗位调研和投递规划。信息差本身就是优势，早发现早行动。

### 投递节奏

- **主攻 3–5 家目标公司**，深度研究其技术体系和面试风格
- **同步投递 10–15 家备选**，维持面试手感
- **内推优先**：LinkedIn、脉脉、校友群找在职员工，内推简历直接进池，比官网投递快2–3周

## 五、面试准备

### 技术岗三件套

| 环节 | 重点 |
|------|------|
| 简历项目深挖 | 每个项目能讲清"背景→方案→难点→结果"，至少30分钟 |
| 算法/编程 | LeetCode 中等 100题打底，重点：二叉树、DP、双指针、BFS |
| 专业知识 | 根据岗位定向复习，如ML八股、系统设计、SQL |

### 非技术岗三件套
- 行业认知：公司在做什么、竞品是谁、你能带来什么价值
- 结构化表达：STAR法则（情境→任务→行动→结果）
- 反向提问：展示思考深度，而非"没有问题"

---

## 六、关键心态

**不要等"准备好了"再投。**

大多数人永远觉得自己没准备好。实际上，面试本身就是最好的练习。

- 早投早试错，早调整策略
- 拿到第一个 offer 后心态会完全不同
- 不要只盯大厂，中型公司的成长空间往往更大
