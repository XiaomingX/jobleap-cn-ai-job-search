# 国内大厂技术面试真题库 (阿里、腾讯、字节、美团、百度)

## 一、阿里巴巴 (Alibaba)
- **侧重点**：Java 并发、中间件（RocketMQ, Dubbo）、架构设计、高可用。
- **真题摘录**：
    1. 分布式事务的最终一致性如何实现？
    2. JVM 调优中，如何定位内存泄漏？
    3. 谈谈你对双 11 高并发场景下限流熔断的理解。

## 二、腾讯 (Tencent)
- **侧重点**：C++ 底层、网络协议（TCP/IP）、Linux 内核、海量数据处理。
- **真题摘录**：
    1. TCP 状态机中的 TIME_WAIT 为什么是 2MSL？
    2. 如何在 100 亿个 URL 中找到重复项？
    3. 解释一下协程的原理以及与线程的区别。

## 三、字节跳动 (ByteDance)
- **侧重点**：算法极其严格（LeetCode Hard）、系统设计、业务响应速度。
- **真题摘录**：
    1. 手写 LRU 缓存。
    2. 设计一个短域名生成器 (URL Shortener)。
    3. 如何保证 Feed 流推送的实时性与一致性？

## 四、美团 (Meituan)
- **侧重点**：MySQL 深度优化、Redis 缓存设计、O2O 业务场景。
- **真题摘录**：
    1. MySQL 聚簇索引与非聚簇索引的区别。
    2. Redis 的大 Key 问题如何处理？
    3. 谈谈美团外卖配送调度中的经纬度索引优化。
